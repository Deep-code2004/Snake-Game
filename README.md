# Snake Game

This is a simple Snake Game built using Python and the `pygame` library. The game includes features like background music, sound effects, a high score tracker, and a restart option.
This Game is a simple implementation of the classic Snake Game The Snake Game can move up ,down, left , right and the goal  is to eat the food while avoiding collisions with the wall or itself 
## Features

- **Classic Snake Gameplay**: Control the snake to eat food and grow longer.
- **Background Music**: Plays continuously during the game.
- **Blast Sound Effect**: Plays when the snake dies.
- **High Score Tracker**: Keeps track of the highest score across sessions.
- **Restart Option**: Restart the game by pressing Enter after the game is over.
- **Customizable**: Easily modify colors, screen dimensions, and other settings.

## Prerequisites

Make sure you have Python installed on your system. You also need to install the `pygame` library.
